## 🚀 Overview

EduSpark is a modern SaaS-based EdTech platform designed as an **AI-powered personalized learning ecosystem**, combining online education, adaptive intelligence, gamification, and mentorship into one immersive product.

Unlike traditional course websites, EduSpark functions as a **student operating system**, where learners can enroll in premium courses, track progress, interact with mentors, solve doubts using AI, and receive personalized learning experiences.

## 💡 Why This Project
EduSpark explores what the future of education could look like when:
- AI acts as a personal tutor
- Learning becomes adaptive
- Progress feels game-like
- Students use an operating system, not just courses

This project was built as both a **product concept** and **frontend SaaS case study**.
